### describe, beforeAll, afterAll, beforeEach, afterEach 사용법

- describe

  안에 있는 모든 테스트를 테스트 하겠다.

- beforeAll

  test들이 모두 다 시작하기 전에 실행되는 부분을 의미

- afterAll

  test들이 모두 실행되고 난 후에 실행되는 부분

- beforeEach

  각 test들이 실행되기 전에 실행되는 부분을 의미

- afterEach

  각 test들이 실행된 이후 실행되는 부분을 의미
